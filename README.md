# OriginNoMigrate

## Instructions

### [Download the latest release](https://github.com/Dyvinia/OriginNoMigrate/releases)

1. Install Origin v10.5.118.52644: https://download.dm.origin.com/origin/live/OriginSetup.exe
2. Run OriginNoMigrate
